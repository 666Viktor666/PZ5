# PZ5

## Bank Deposit Calculator

Простой калькулятор вкладов на Java с использованием циклов while и for.

### Функциональность
- Расчёт сложных процентов по вкладу
- Подробный помесячный вывод
- Реализация через циклы while и for

### Запуск
- javac src/Main.java
- java src/Main
